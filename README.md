# Deployment Steps

## Clone Repository
```sh
git clone https://github.com/Collin-Varisco/LA-College-Movers.git
```
## Install Node.js Dependencies
```sh
cd la-college-movers
```
```sh
npm install
```
## Start The Server
```sh
npm start
```

# Access Website
[http://localhost:8080](http://localhost:8000)
